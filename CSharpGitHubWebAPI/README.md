
# Scenario:

A Web API for Popular Repos on GitHub

## Scenario Description:

- A Web API that returns a list of repos on GitHub,
- Http Client with RestSharp,
- Filtering and Sorting on Frontend using JavaScript,
- The results are shown in a paginated way,
- The API should be able to receive any request from any domain.

## Technologies Used:

- C# 9.0,
- .NET 8.0,
- RestSharp,
- CORS,
- Swagger,
- Postman.
